Frigg

Calculate how to evenly in- or de-crease the number of masks.

Input the number of masks you have on the top line and the number of masks you want to end with in the second line.

Numbers are adjusted with the / buttons. Increment by 1/10/50 with the small/medium/large buttons.

The result can contain either a single pattern to repeat or two patterns that you need to mix.
